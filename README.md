# Diploma_2015


 * client.py -> скрипт, който подава данни на порт, подаден като аргумент от командния ред. Debug because #nohardware
 * gameEngine.py -> "ядрото" на приложението, по-голямата част от нещата са там.
 * callScript.py -> скрипта, който трябва да се викне при искане да се играе. Дава опции за неща

 - python/ -> Код за сайта - от него се слагат игри, и се теглят. Сайта трябва да седи на localhost:80 (var/www/html при правилно setup-нато apache)
 - old_stuff/ -> Ненужни файлове, използване в периода на разработка на софтуерния продукт
 - games/ -> Директорията с игрите, свалени на конкретната игрова конзола
 - diplomna_documentation/ -> Съдържа електронния вариант на книжното тяло на дипломната работа
